# ps-windows-defender

ps-windows-defender is a PowerShell script meant to manage several Windows Defender functions in a breeze.

## Installation

Nothing fancy here! Just need a Windows device and an open PowerShell Terminal. 

## Usage

```bash
cd C:\(install location...) 
.\NevinsDefender.ps1
```
